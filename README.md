# Rock Paper Scissors!!!

A simple Python game of Rock, Paper, Scissors where the user plays against the computer.

## How to Play
- Enter:
  - `0` for Rock
  - `1` for Paper
  - `2` for Scissors
- Game is played for 5 rounds
- Final score is shown at the end
